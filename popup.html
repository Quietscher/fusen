fusen – Sticky notes that stick

Welcome to fusen, a chromium extension that adds sticky notes to any webpage, that stay.

Start either by clicking the button below or by using the keyboard shortcut.

Keyboard Shortcuts

Ctrl + ⌘/⊞ + x : Delete Note
Ctrl + ⌘/⊞ + y : New Note
Ctrl + ⌘/⊞ + < : Temporarily Close Note

Sticky Notes Settings

How does fusen work?

With fusen, sticky notes are matched to the current page by the domain and the pathname.

The input defines the percentage of the pathname that must match to show the sticky note.

At 0, only the domain must match.
